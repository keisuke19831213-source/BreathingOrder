# Breathing Order

> 秩序は静止ではなく、呼吸する生命体である。

## 概要

p5.js を用いたジェネレーティブアート作品。
幾何学的な円構造が、まるで生き物のように静かに呼吸する様子を表現しています。

## ビジュアル構造

- **8 層の同心円リング** — 各リングは等間隔に配置された点群で構成される純粋な幾何学形状
- **ポリゴン輪郭** — 各リングの点を結ぶ淡い多角形の線（秩序の骨格）
- **スポーク** — 中心から外縁へと伸びる 6 本の極薄の軸線
- **中心グロウ** — 呼吸に合わせて微かに明滅する光の核

## 動きの設計

| 要素 | 説明 |
|------|------|
| 呼吸周期 | sin() による約 15 秒/呼吸（60fps 基準）|
| 揺らぎ | Perlin noise で周期を微妙に乱し「完全でない」呼吸を実現 |
| 位相差 | 外側リングほど遅れて膨張し、さざ波のような伝播を表現 |
| 回転 | 約 8 分で一周するほぼ知覚不能なゆっくりとした回転 |
| 点の揺れ | 各点が独立した noise シードで位置・サイズ・輝度を揺動 |

## カラーパレット

```
背景:  #000000（純黒）
要素:  rgba(255, 255, 255, α)（白・透明度のみ変化）
```

## 実行方法

```bash
# ブラウザで index.html を開くだけ
open index.html
```

またはローカルサーバー（ファイルアクセス制限がある場合）:

```bash
python3 -m http.server 8000
# → http://localhost:8000 をブラウザで開く
```
